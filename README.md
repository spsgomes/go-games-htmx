# Installation

### Prerequisites
- Install [Golang](https://go.dev/dl/). This project was developed in version `1.24.0`, on `windows/amd64`.

### Clone the repository
```sh
git clone https://github.com/spsgomes/go-games-htmx.git
cd go-games-htmx
```
